Documentação da API - ANEEL Reclamações

Bem-vindo à documentação da API de concessionárias. Esta API permite acessar informações sobre concessionárias de energia elétrica, com base em diferentes critérios.

1. Buscar Todas as Concessionárias

Descrição: Retorna uma lista de todas as concessionárias cadastradas.

URL: /api/concessionarias

Método: GET

URL: /api/concessionarias/nome/:SigAgente

4. Buscar Concessionárias por Sigla de UF

Descrição: Retorna uma lista de concessionárias com base na sigla do estado (UF) fornecida.

URL: /api/concessionarias/uf/:SigUF

Método: GET
